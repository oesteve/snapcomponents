## Folder structure

```
/assets/admin
│
├── featureA/                     # Each feature at root level
│   ├── components/               # Components specific to this feature
│   ├── hooks/                    # Hooks specific to this feature
│   ├── index.ts                  # Entry point for exporting feature elements
│   └── lib/                      # non react realted functions modules
│
├── featureB/
│   ├── components/
│   ├── hooks/
│   └── index.ts
│
```
